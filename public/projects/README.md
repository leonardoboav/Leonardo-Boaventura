# Prints dos projetos

Coloque aqui os prints (demos em imagem) usados na seção de portfólio
(`src/components/projects-section.tsx`).

## Especificação

- **Proporção 16:9** (ex.: 1600×900 ou 1920×1080).
- Capture em **viewport de 1440px** de largura, para manter consistência entre os prints.
- Nomeie cada arquivo igual ao campo `image` definido em `src/lib/projects.ts`:
  - `realconnect.png`
  - `bluesky.png`
  - `washclub.png`

Formatos recomendados: `.png` ou `.webp`. Até os arquivos reais existirem, os cards
apenas mostram a área da imagem vazia — é só soltar os prints aqui com os nomes acima.
